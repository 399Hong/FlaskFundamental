## Installation

**Installation via requirements.txt**

```shell
$
$ py -3 -m venv venv
$ venv\Scripts\activate
$ pip install -r requirements.txt
```

## Execution

**Running the application**

````shell
$ flask run
````

http://127.0.0.1:5000/
